GPP is mainly aimed at public authorities (government agencies / offices / institutions) that are obligatoire due to statutory requirements to publish their spatial data and want to make a offer of information.

The following open source components are used: Geonetwork, GeoExt (ExtJS, OpenLayers), GeoServer, PostGIS.

FAQ

Why using open source?
Big community, full source code control.
Do i need programming skills?
No, GPP is fully configurable via property files. But you can add custom code if you like.
